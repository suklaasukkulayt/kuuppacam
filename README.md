# kuuppacam  
Made for stardance competition, more info in my devlogs there  
I did this to learn html (I made kuuppacam-old and styling)  
This is a DIY surveillance camera that uses the IP camera app and an old android phone (and a 3d-printer(not required)).  
Features: live video, live audio, recording(made with github copilot), easy to change ip(made with github copilot)  

If you want hardcoded ip address just change lines 123 and 124. You still have to submit something for it to load(it can be anything)  

The app is made with appcreator24 and html code made in vs code  
it's not the best but it works, for example as a baby monitor  

no ai used in the kuuppacam-old.html version  
Github copilot is used in the new version to add recording and easy ip changing.
